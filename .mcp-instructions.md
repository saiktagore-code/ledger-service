# Event Ledger MCP Instructions

This project uses a simple developer flow for local validation and CI checks.

## Prerequisites

- Java 21
- Maven 3.9+
- Docker (optional for local compose)

## Local commands

Run tests and build both services locally:

```bash
./run-local-checks.sh
```

Run gateway service build only:

```bash
cd gateway-service
mvn -B clean package
```

Run account service build only:

```bash
cd account-service
mvn -B clean package
```

## Sonar and CodeQL

For GitHub Actions, set these secrets in the repository:

- `SONAR_TOKEN`

### Sonar

The workflow path is `.github/workflows/sonar.yml`.
It runs Sonar analysis for both services against SonarCloud.

### CodeQL

The workflow path is `.github/workflows/codeql-analysis.yml`.
It runs standard Java CodeQL analysis.

## Docker Compose

Start both services locally with Docker Compose:

```bash
export ACCOUNT_SERVICE_API_KEY=supersecret
docker compose up --build
```

## Notes

- The repository currently does not include a parent aggregator POM.
- Use the per-service Maven commands above for local builds.
